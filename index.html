🚀 Quick Start

For Coordinators:

1. Open the Web Interface (link above)

2. Upload Excel file to SharePoint/Teams

3. Share with your team

4. Use the dashboard to monitor progress

For Agents (Elías, Gabriela, Jacqueline):

1. Open Excel file in SharePoint

2. Click your name sheet

3. Log daily activity (date, task, hours, status)

4. Save - metrics update automatically

📋 System Architecture

Excel (Data) ← SharePoint ↓ Web Interface ← GitHub Pages ↓ Dashboard (Real-time)

🎯 Weekly Workflow

Friday EOD: Fill "Weekly Plan" in Excel with all tasks

Monday-Friday: Agents log daily activity in their sheets

Daily: Check Web Dashboard for real-time progress

Thursday 5 PM: Create weekly report from Dashboard metrics
