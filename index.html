🚀 Quick Start

For Coordinators:

1. Click "Open Dashboard" above to view the web interface

2. Click "Refresh Data" button to load tasks from Excel

3. Monitor team progress in real-time

4. Share updated Excel with team members

For Agents (Elías, Gabriela, Jacqueline):

1. Download the Excel file (button above)

2. Open your personal sheet

3. Log daily activity (date, task, hours, status)

4. Upload updated file back to GitHub

📊 System Architecture

Excel File (GitHub) ↓ HTML Interface reads Excel ↓ XLSX.js library processes data ↓ Dashboard displays in real-time

✨ Key Features

✅ Real-time Sync - HTML reads Excel automatically

✅ Kanban Board - Visual task layout by day

✅ Auto Metrics - Completion %, hours, priority

✅ Team Progress - Individual agent statistics

✅ GitHub Hosted - No servers, 100% GitHub

✅ Refresh Button - One-click data update

📋 Weekly Workflow

Friday EOD: Edit Excel → Add all tasks for next week → Upload to GitHub

Monday-Friday: Agents log daily activity in their sheets → Upload updates

Daily: Coordinator opens dashboard → Click "Refresh Data" → Monitor progress

Thursday 5 PM: Screenshot dashboard metrics → Create weekly report

🔄 How to Update Data

Option 1: Edit Excel locally

1. Download Excel from GitHub 2. Edit in Excel (add/update tasks) 3. Save file 4. Upload back to GitHub 5. In dashboard: Click "Refresh Data"

Option 2: Edit directly in GitHub

1. GitHub > CS_Team_Weekly_Tracker.xlsx 2. Click edit button 3. Make changes 4. Commit 5. In dashboard: Click "Refresh Data"
