📋 Complete Daily Workflow

For Agents (5 minutes/day):

  1. Open system and click "Daily Execution"
  2. Select your name from dropdown
  3. For each assigned task:
    • Enter hours worked today
    • Enter subtasks completed
    • Select status (Pending/In Progress/Completed)
    • Add comments about blockers or progress
  4. Click "Save Daily Activities"
  5. Data saved automatically, nothing else needed

For Coordinators (Real-time monitoring):

  1. Monitor "Weekly Planning" - see all tasks per agent
  2. Watch "Kanban" - see progress bars updating daily
  3. Review "Comparison" - see planned vs executed totals
  4. Friday 5 PM: Full weekly report ready with real data

💡 Practical Example

Task: "Update 20 FAQs of Payments Section" (10 hours planned for Gabriela)

What Happens:

  • Monday: Gabriela logs 2.5h, 5 subtasks → Kanban shows 25% progress
  • Tuesday: Gabriela logs +2h, +4 subtasks → Kanban shows 45% progress
  • Wednesday: Client requests 10 more FAQs → Gabriela updates work
  • Thursday: Gabriela logs +3h, +6 subtasks → System detects scope change
  • Friday 5 PM: Coordinator sees: Planned 35h total, but gained 10 extra FAQs
  • Report: "Scope increased by client request. Gabriela delivered 30 FAQs in 28h"
