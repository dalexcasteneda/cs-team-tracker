📋 Complete Daily Workflow

Coordinator (Monday 9 AM)

  1. Open system and click "Add Tasks"
  2. Plan all tasks for the week:
    • Select agent (Elías, Gabriela, Jacqueline)
    • Select task type (Ticket Support, FAQ, etc)
    • Select day (Monday-Friday)
    • Enter hours
    • Set priority and status
    • Add description and comments
  3. Click "Add Task" for each one
  4. Tasks appear in "Weekly Planning" automatically
  5. Tasks appear in "Kanban" automatically

Agents (Monday-Friday, 5 min/day)

  1. Open system and click "Daily Activity"
  2. Select their name
  3. For each task they worked on:
    • Enter time worked today
    • Enter subtasks completed
    • Select status (Pending/In Progress/Completed)
    • Add comments about blockers or progress
  4. Click "Save Activities"
  5. Data updates in Kanban automatically

Coordinator (Friday 5 PM)

  1. Open "Kanban" tab to see overall progress
  2. Open "Comparison" tab to see detailed analysis
  3. View Planned vs Executed hours
  4. See variance per task (highlighting overload)
  5. Generate weekly report with real data
